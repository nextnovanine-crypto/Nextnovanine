ANLEITUNG
01
Einloggen
Logge dich mit deinem Flowise Account ein wenn du dazu aufgefordert wirst.
02
Neuen Chatflow erstellen
Klicke auf "Add New" um einen neuen KI Agenten zu erstellen. Wähle zwischen Chatflow und Agentflow.
03
Nodes verbinden
Ziehe Nodes per Drag & Drop auf die Canvas und verbinde sie miteinander. Trigger → LLM → Output.
04
System Prompt setzen
Klicke auf den LLM Node und gib deinen System Prompt ein — das ist die Persönlichkeit deines Agenten.
05
Speichern & testen
Klicke auf "Save" und teste deinen Agenten direkt im Chat-Fenster rechts unten.
06
Einbetten
Klicke auf "</> Embed" um den Code zu kopieren und deinen Agenten in deine Website einzubinden.
